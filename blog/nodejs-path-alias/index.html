  2. Node.js 配置别名 alias 的两种方法

module-alias 方案应用#

  1. 安装 module-alias
yarn add module-alias
  1. 在 src/index.ts 中注册 add module-alias
import 'module-alias/register';
  1. 在项目 package.json 中新增别名配置
"_moduleAliases": {
"@": "./bin"
},
